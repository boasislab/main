1. 나자식물 vs 피자식물 — 기본 분류

핵심 구분: 나자식물 = 씨앗이 겉(노출)에 있다 → 열매가 없다.
피자식물 = 씨앗이 씨방 안에 있다 → 열매가 있다.

나자식물

  • 꽃잎·꽃받침이 없고, 씨방(자방)도 없음 → 열매를 맺지 않음
  • "열매처럼 보이는 것"은 사실 구과(솔방울) — 솔방울 안에 씨앗이 날개를 달고 있음
  • 우리나라 나자식물: 소나무, 잣나무, 전나무, 주목, 향나무, 은행나무 등
  • 솔방울(구과)은 열매가 아니라 씨앗의 집합체
  • 우리나라 나자식물 약 50종
소나무의 생식기관 명칭:
수생식기관 = 소포자낭수 (송화가루, 수꽃)
암생식기관 = 종구 (붉은색 = 암꽃 = 둥근종자 = 둥근씨앗 = 과거 구과)
꽃가루(화분) 안에: 관핵 1개 + 정자 2개 = 총 3개
수술머리 = 약(葯). 약에는 주머니 4개.
나자식물 → 암생식기관 = 대포자 / 피자식물 → 암생식기관 = 밑씨(배주)
소나무: 나자식물. 꽃×, 열매× / 솔방울 = 씨앗 = 종구
솔방울 안에 여러 개의 교각이 있고, 교각 위에 씨앗이 2개가 날개를 달고 놓여있음.
수술머리는 약이라고 함. 약에는 주머니 4개. 꽃가루 안에 관핵 1개, 정자 2개 총 3개 있음.

은행나무 — 나자식물 특이사항

  • 자웅이주(암수딴그루) — 암나무에서만 은행(종자)이 열림
  • 나자식물이므로 꽃잎·꽃받침 없고, 씨방도 없음 → 열매가 아니라 종자 자체가 열림
  • 은행 종자: 말랑말랑한 외종피 → 딱딱한 내종피 → 씨앗 순서
  • 은행나무 암꽃은 밑씨 = 배주
  • 은행나무 수그루: 소포자낭수(피자식물의 꽃가루주머니)
넓은잎삼나무(중국): 암수한그루. 암술×, 수술×, 꽃잎× → 꽃×, 종자○
종구: 암생식기관 / 소포자낭수: 수생식기관

2. 소나무과 (나자식물 대표)

소나무속(Pinus) 대표 수종

  • 소나무 (Pinus densiflora) — 적송
  • 잣나무 (Pinus koraiensis) — 홍송. "koraiensis" = 우리나라(Korea) 소나무라는 학명
  • 금강소나무 — 강선(剛鮮)이라고도 함. 태백·삼척 일대. 줄기가 통직(通直). 황장목(黃腸木)이라 불림
  • 전나무(Abies): 소나무과이지만 전나무속(Abies)에 속함 — 소나무속 아님
수목원 기억법 — 메대낙호: 메타세쿼이아 대생(마주나기), 낙우송 호생(어긋나기) → 잎, 잔가지 구분
산호초대: 산초나무 호생(어긋나기), 초피나무 대생(마주나기) → 잎, 줄기에 달린 가시 구분
초피나무 = 젠피 = 계피 = 전피. 추어탕에 넣음. 실제 약 효용 좋음. 열매 안의 씨 빼고 껍질 갈아 먹음.

3. 세계 3대 미수 · 식물 분류 체계

세계 3대 미수(美樹)

  1. 히말라야시다 (개잎갈나무, 소나무과)
  2. 아라우카리아 — 우리말: 남양삼나무. 적도 인근 남쪽 지역 자생. 우리나라에 자생하지 않아 삼나무와 구별 필요
  3. 금송(金松) — 소나무과가 아니라 낙우송과(Taxodiaceae). 시험 자주 출제!

식물 분류 체계

분류 단계 (작은 것 → 큰 것): 종 < 속 < 과 < 목 < 강 < 문 < 계 < 역 (p.238)
  • 과(科) 안에는 나무와 풀이 함께 있는 경우가 많음
  • 나무만 있는 과: 소나무과 등 / 풀만 있는 과: 벼과 등
  • 우리나라 식물 종수: 약 5,000종 (나자식물 약 50종, 피자식물 대부분)
  • 전 세계 식물 종수: 약 30만~35만 종 (나무는 약 8만 종)

4. 식물의 기관 — 영양기관 · 생식기관

영양기관 3가지 = 잎 · 줄기 · 뿌리
생식기관 3가지 = 꽃 · 열매 · 씨앗(종자)
생식기관 중 식물에서 가장 중요한 것은 꽃. 꽃에서 열매와 씨앗이 만들어짐.

줄기의 변형 — 인경(鱗莖)

  • 양파·상사화 등의 뿌리처럼 보이는 구조는 사실 뿌리가 아님
  • 인경(鱗莖) = "비늘 줄기". 줄기가 변형된 것
  • 양파를 껍질째 벗기면 — 바깥 껍질 = 잎(엽)이 변형된 것, 안쪽 흰 덩어리 = 줄기(경)가 변형된 것
  • 석산(꽃무릇): 인경(비늘줄기). 귀화식물

잎의 구조

  • 잎 = 잎자루(엽병) + 잎몸(엽신)
  • 잎살(엽육): 잎맥과 잎맥 사이
  • 잎몸 = 잎맥(엽맥) + 잎살(엽육)
  • 잎맥: 관다발의 통로 역할
  • 엽저: 잎의 아래쪽, 잎자루와 만나는 부분
  • 엽두 = 엽정 = 엽선: 잎의 머리, 끝부분(꼭대기)

잎눈 (엽아)

  • 눈에서 잎이 나옴. 눈이 분열해서 잎이 나옴
  • 잎이 나오면 눈은 항상 잎 위에 있는 것임

엽병내아(잎자루 안에 눈 있음) — 잎자루 속눈 3가지

  1. 황벽나무
  2. 플라타너스 (= 버즘나무)
  3. 때죽나무 = 쪽동백나무 (단, 쪽동백나무는 때죽나무과 때죽나무속 / 동백나무는 차나무과 차나무속)

깃꼴겹잎 · 잎 배열

  • 깃꼴겹잎: 하나의 잎이 여러 개의 작은 잎(소엽)으로 나뉘고, 그 배열이 깃털처럼 생긴 잎
  • 말오줌때: 잎이 깃꼴 겹잎. 잎이 더 펴지면 광택 있음
  • 메타세쿼이아·낙우송: 복엽× 겹잎× 하나의 잎임

5. 꽃의 구조

꽃의 4요소 = 암술 + 수술 + 꽃잎 + 꽃받침
4가지를 모두 갖춘 꽃 = 갖춘꽃(완전화)
꽃잎과 꽃받침이 없고 암술·수술만 있는 경우도 있음 (예: 소나무 꽃 = 안갖춘 꽃)

씨방(子房)과 열매·씨앗

씨방 → 열매 / 밑씨(배주) → 씨앗(종자)
암술 아래에 씨방(자방) 있음. 씨방이 성숙하면 → 열매가 됨.
씨방 안에 밑씨(배주) 있음. 밑씨가 성숙하면 → 씨앗(종자)이 됨.

수술의 구조

  • 수술 = 꽃밥(약, 葯) + 수술대(화사)
  • 꽃밥(약) 안에 꽃가루주머니 4개, 주머니 안에 꽃가루(화분)
  • 꽃가루(화분) 안에는 정자 2개 들어 있음

암술의 구조

  • 암술 = 암술머리(주두) + 암술대(화주) + 씨방(자방)
  • 개나리처럼 암술이 긴 꽃 = 장주화(長柱花)
  • 암술이 짧은 꽃 = 단주화(短柱花)
  • 한 개체에 장주화·단주화가 모두 있어 섞이지 않고 수분되는 것 = 이형예현상

꽃의 구조 도해

피자식물 꽃 구조도 (Gnote p.274):
암술머리 → 암술대 → 씨방(밑씨/배주) — 이것이 암술
꽃밥(꽃가루주머니 4개) + 수술대/화사 — 이것이 수술
꽃잎 · 꽃받침 · 화탁(꽃턱)

화피(花被)

  • 꽃잎과 꽃받침이 구분 안 될 때 = 화피
  • 대표: 목련류. "목련: 공룡이전 나옴. 향 없음(나비, 벌 없었을 때). 매개체–딱정벌레. 벼과식물=원시적."

총포 · 총포편

  • "꽃"이 하나라면 그냥 "포"임
  • 꽃이 여러 개 = 총포 (모일총을 써서 "총"이 붙음)
  • 각각의 조각 = 총포편 (포엽)
  • "포" = 꽃턱의 잎
  • 꽃잎과 꽃받침을 지탱해주는 것 = 꽃턱(화탁)

6. 꽃의 성(性) 분류

양성화(兩性花): 한 꽃에 암술+수술 모두 있음 — 전체 식물의 약 2/3 (70%)
단성화(單性花): 암꽃 또는 수꽃만 (암술만 or 수술만)
  ㄴ 자웅동주(雌雄同株, 암수한그루): 암꽃·수꽃이 한 나무에 → 소나무, 옥수수
  ㄴ 자웅이주(雌雄異株, 암수딴그루): 암나무·수나무 따로 → 은행나무, 생강나무, 뽕나무 일부

소나무의 성

  • 소나무는 암수한그루(자웅동주). 한 나무에 암꽃(윗부분)과 수꽃(아랫부분)이 따로 있음
  • 꽃잎·꽃받침 없음 → 갖추지 못한 꽃(불완전화)

은행나무의 성

  • 은행나무는 자웅이주(암수딴그루). 암나무에서만 은행(종자)이 열림

무화과의 성

  • 무화과(Ficus carica): 꽃이 안에 숨어 있어 겉으로 안 보임
  • 안쪽에 암꽃과 수꽃이 따로 있음
  • 무화과좀벌이 작은 구멍으로 들어가 수분. 벌은 안에서 나오지 못하고 죽음
  • 자성양성이주 — 암꽃+양성화가 다른 그루

옥수수의 성

  • 옥수수는 자웅동주(암수한그루)
  • 수꽃은 위(이삭대)에, 암꽃은 옆(수염=암술머리)에 있음
  • 율무도 암수한그루

이팝나무의 성

이팝나무: 수꽃양성화딴그루

왕벚나무 꽃 구조

  • 꽃잎 5개, 꽃받침 5개 (5의 배수) / 수술 10개 (5배수 × 2)
  • 암수한꽃 (양성화)
  • 피자식물 쌍떡잎: 5배수·4배수
  • 단자엽: 3배수 (외떡잎식물)

7. 식물의 성(性) 상세 분류표

분류설명대표 수종
자웅동주
(암수한그루)		 암꽃과 수꽃이 같은 그루에 핌 소나무, 흑백나무, 으름덩굴, 느티나무, 신갈나무, 박달나무 등
자웅이주
(암수딴그루)		 암꽃과 수꽃이 각각 다른 그루에 핌 은행나무, 비자나무, 주목, 생강나무, 산뽕나무, 감나무, 옻나무, 초피나무 등
웅성양성동주 수꽃+양성화가 같은 그루 모과나무, 자귀나무, 단풍나무, 호박, 수박, 파파야 등
자성양성동주 암꽃+양성화가 같은 그루 음나무, 팽나무, 조독나무 등 / 민들레 (암꽃양성화한그루)
잡성동주 양성화·수꽃·암꽃이 같은 그루 은란공, 자리나무, 병조희풀
웅성양성이주 수꽃+양성화가 다른 그루 다래나무, 왕머루, 후피향나무, 부게꽃나무, 들메나무 등
자성양성이주 암꽃+양성화가 다른 그루 무화과나무, 천선과나무, 모란, 애기모탑 등
잡성이주 양성화·수·암이 다른 그루 은단중, 장구밤나무, 땃두릅나무 등
다접자웅이주 양성화+암꽃이 같이 피는 그루와 양성화+수꽃이 같이 피는 그루가 각각 있음 —
수꽃양성화딴그루 수꽃양성화가 딴그루에 핌 이팝나무 / 시무나무

병조희풀

8. 쌍떡잎식물 vs 외떡잎식물

꽃잎 수로 구분:
쌍떡잎식물(쌍자엽식물): 꽃잎 4~5개(4배수·5배수) — 장미, 버드나무, 무궁화 등
외떡잎식물(단자엽식물): 꽃잎 3~6개(3배수·6배수) — 백합, 난초, 벼, 옥수수 등
외떡잎식물의 뿌리: 수염뿌리 / 쌍떡잎식물의 뿌리: 곧은뿌리(주근)
비고식물: 양성화(다수) / 대나무: 양성화 / 산벚나무: 지리산 고산대 / 상수리나무: 암수한그루

목련 — 쌍자엽이지만 3배수 예외

  • 목련: 쌍자엽으로 4배수~5배수이어야 하나, 3배수로 단자엽에 유사
  • 일본목련: 화피 12개 = 3의 배수 (단자엽 특성)

쿠페아

  • 꽃잎 6개. 3자엽. 단자엽

9. 국화과 꽃 구조 — 두상화서(頭狀花序)

국화과 꽃의 2가지 소화:
통꽃(관상화, 筒狀花): 안쪽 꽃 — 암수술이 통 안에 있음 → 양성화
혀꽃(설상화, 舌狀花): 가장자리 꽃 — 수술이 없음 → 암꽃 or 중성화

민들레: 통꽃만 있고, 모두 양성화 / 머리모양 꽃차례 = 두상화서
국화: 통꽃(양성화) + 혀꽃(암꽃 or 중성)이 함께 있음
국화과 — 꽃받침 없음

민들레 상세

  • 혀꽃○ 통꽃× 양성화 (암꽃양성화한그루)
  • 꽃받침처럼 보이는 것 = 총포. 각각을 총포편이라고 함
  • 총포가 2겹: 겉에 있는 총포가 늘어지면 서양민들레(돌기 없음)
  • 우리나라 민들레: 겉에 있는 총포에 돌기 달려 있음 = 늘어지지 않음
  • 민들레씨: 갓털(수많은 꽃들이 각각의 씨방이 있고, 각각의 열매가 하나의 열매로 보임) = 관모
  • 털은 멀리 퍼뜨리기 위해 있음. 각각의 씨앗, 각각의 열매. 홀씨× (포자만 홀씨임)

국화과

  • 지칭개: 뜨거운 양지에 있음. 지친개처럼 보임. 보라꽃. 진딧물 많음=단물 많음. 괴산 특산 나물로 먹음
  • 국화·엉겅퀴 양성화(국화) / 양성화 — 영경구

10. 미나리아재비과(Ranunculaceae)

미나리아재비과의 특징: 꽃잎이 없고 꽃받침이 꽃잎 역할을 함

바람꽃 종류

  • 하얗게 보이는 것이 꽃받침 (꽃잎이 아님). 꽃잎의 수는 종류마다 5~12개

  • 꿩의 바람꽃: 꽃밭의 가장 많음 (10~15m)
  • 홀아비 바람꽃: 1줄기 1꽃대
  • 번스 바람꽃: 유일하게 꽃잎이 있음
  • 너도바람꽃: 꽃잎이 있음 — 우리나라에 2종(오대산 너도바람꽃, 연산 너도바람꽃)
  • 최리 바람꽃

매발톱 · 세미매발톱

  • 매발톱
  • 세미매발톱 = 개구리발톱 — 열매가 개구리발톱 모양. 꽃잎 4개. 3월 말~4월 초 꽃핌. 세미=작다 뜻

피나물 vs 매미꽃

11. 상사화류 · 특산종 · 희귀종

상사화 종류

  • 상사화류는 인경(알뿌리, 실제론 줄기 변형)을 가짐
  • 진노랑상사화: 내장산(백양사 계곡), 고창 선운사에만 자생. 우리나라 법정 보호식물 90종 중 하나. 멸종위기 야생식물 2급
  • 북노랑상사화: 부안. 우리나라 특산종(고유종)
  • 위도상사화: 자생종. 임경, 경엽
  • 석산(꽃무릇): 사찰 경내에 많이 심어 불화(佛畵)의 분위기를 냄. 귀화식물
특산종 = 고유종 = 우리나라에만 있는 종 (같은 말)
귀화종: 외국에서 들어와 야생화된 종
자생종: 우리나라에 원래부터 자라는 종

우리나라만 있는 6개 속 (고유속)

  1. 매미꽃속: 매미꽃 (피나무는 피나무과 피나무속으로 아님)
  2. 미선나무속: 1원종, 3개 품종(상아미선·분홍미선·푸른미선), 1변종(둥근잎미선). 물푸레나무과 미선나무속. 줄기가 동물꼬리, 열매가 부채 모양(꼬리미, 부채선)
  3. 제주고사리삼속: 제주고사리삼 (제주 동백동산)
  4. 금강초롱꽃속: 청사초롱 (① 금강초롱꽃 ② 검산초롱꽃—금강산 가야함)
  5. 금강인가목속: 금강산 암벽에만 붙어있음
  6. 더구기름나물속: 석회암지대. 강원도 정선 더구리 동네 (원빈, 이나영)
※ 개느삼속은 최근 만주쪽에서 발견되어 제외됨. 개느삼=너삼

개느삼

1. 나무와 풀의 구분 — 나이테(연륜)

나무와 풀의 가장 정확한 구분 기준: 나이테(연륜) 유무
나무 → 매년 형성층에서 직경 생장(2차 생장)을 하여 나이테가 1년에 하나씩 생김
풀(초본) → 2차 생장(직경 생장)을 하지 않아 나이테가 없음
야자수, 대나무 → 나이테 없음 → 식물학적으로 풀(초본)

잘못된 구분 기준

  • "풀은 키가 작고 나무는 키가 크다" — 예외: 암매(돌매화나무) 한라산 꼭대기 바위틈 자생, 전체 크기 5~10cm지만 분명한 나무
  • "겨울에 지상부가 죽느냐" — 예외: 도깨비고사리·꼬리고사리처럼 겨울에도 잎이 살아있는 상록성 풀 있음
  • "형성층 유무" — 예외: 해바라기처럼 풀이면서도 형성층이 있는 식물 있음
대나무의 분류: 용도학적으로는 나무처럼 쓰이지만, 식물학적(분류학적)으로는 풀(초본).
이유: 대나무 속이 비어 있어 나이테(연륜)가 없기 때문.

나이테 = Annual Ring(애뉴얼 링) = 연륜(年輪)

  • 나이테 간격이 고르면 좋은 환경, 한쪽이 좁으면 경사지나 스트레스 환경에서 자란 것
  • 춘재(春材): 봄·여름에 자란 밝은 부분 (= 조재, early wood)
  • 추재(秋材): 가을에 자란 짙은 부분 (= 만재, late wood)
  • 가운데 짙은 부분: 심재(心材) — 세포가 죽어있으나 단단함
  • 바깥 밝은 부분: 변재(邊材) — 살아있는 세포

수령 측정 방법

  • ① 심은 기록 확인
  • ② 생장추(increment borer)로 목편 채취 후 나이테 계수 — 뿌리가 가운데 통하지 않을 때, 줄기의 중심을 향해 생장추를 들여넣고 육안으로 나이테 수를 세어 나정. 뿌리에서 발달한 거리를 감안하여 추가 (모를 경우 가슴높이이면 5년 추가)
  • ③ 레지스토그래피(resistograph) — 수령측정기(비파괴검사법). 나무줄기에 침형 센서를 삽입하여 목재밀도에 따른 전기저항 변화값을 이용. 그 자리에서 바로 충재와 부재의 구분을 그래프로 볼 수 있어 편리. 그러나 센서가 닿지 않는 한계로 느티나무류 등 단단한 나무와 거대한 노수의 나이를 측정하기 어려워 실리 이용되지는 못함
  • ④ LiDAR 드론·위성 활용
  • 침엽수는 나이테가 선명하여 바로 계수 가능, 활엽수는 화학 처리로 선명하게 만든 뒤 계수

2. 수목 식별 · 분류 · 명명

용어 정의

  • 식별(동정, identification): 눈앞의 식물이 어떤 종인지 구별하는 것 (일반인)
  • 분류(classification): 식물 간의 서로 가깝고 먼 관계를 따지는 유연관계를 밝혀서 식물의 체계를 세우는 것 = 식물의 족보를 만드는 것 (전문가)
  • 명명(nomenclature): 새로운 종에 ICN(국제식물명명규약)에 따라 공식 이름(학명)을 부여하는 것 (전문가)

APG 분류체계

  • 현재는 DNA 분석에 기반한 APG(피자식물계통군) 분류체계로 단일화
  • 논문 작성 시 어떤 분류체계를 따랐는지 반드시 명기해야 함
  • 국가식물 표준 목록: www.nature.go.kr

분류 단위

분류 단위(크기 순): 계 > 문 > 강 > 목 > 과 > 속 > 종
총 < 속 < 과 < 목 < 강 < 문 < 계 (작은 단위 → 큰 단위)
식물 분류의 기본 단위는 종(種). 종 아래 아종·변종·품종이 있음
예) 소나무: 식물계 > 구과식물문 > 소나무강 > 소나무목 > 소나무과 > 소나무속 > 소나무

영양기관과 생식기관

  • 꽃·열매·종자: 생식기관 (성 있음, 유성생식 관여)

② 호흡: O₂ 흡수, CO₂ 배출 (광합성에 비해 미약)
③ 증산작용: 기공을 통해 수분을 수증기로 배출 → 물의 상승 원동력
광합성·호흡·증산작용 담당 기관 = 기공(잎 뒷면)

  • 측맥(側脈): 주맥에서 좌우로 뻗은 것
  • 턱잎(탁엽): 잎자루 아랫부분 작은 잎 모양 부속물 (버드나무류 등)

5. 꽃과 번식

수분과 수정

  • 수분(受粉): 꽃가루가 암술머리에 붙는 것
  • 수정(受精): 꽃가루관이 자라 밑씨와 세포핵이 결합하는 것
  • 수정 후 씨방이 자라 열매가 됨
  • 수분 방식: 풍매화(바람 — 소나무 등), 충매화(곤충 — 약 80% 이상)

양성화·단성화·중성화

양성화: 한 꽃에 암술·수술 모두 있음 (장미, 매실 등)
단성화: 암술 또는 수술만 있음 (소나무 — 자웅동주+단성화)
중성화(장식꽃): 암술·수술 모두 없음 (불두화)
잡성화: 단풍나무

소나무: 자웅동주(암수한그루)이지만 양성화가 아닌 단성화!
암꽃(위쪽, 빨간색)과 수꽃(아래쪽, 송화가루)이 같은 나무의 서로 다른 가지에 핌
  • 자웅동주(암수한그루): 암꽃·수꽃이 같은 나무에 — 소나무, 살구나무
  • 자웅이주(암수딴그루): 암나무·수나무가 따로 — 은행나무, 버드나무류

번식 방법

  • 유성번식(종자번식): 유전적 다양성 기대 가능, 부모 형질 100% 보장 안 됨
  • 무성번식(영양번식): 삽목·취목·접목 — 모수 형질 100% 유전
  • 접목 시 형성층을 정확히 맞춰야 성공
  • 영양번식: 접목, 취목, 삽목

6. 겉씨식물 vs 속씨식물

속씨식물(피자식물): 씨방 안에 씨 → 진정한 꽃(암술·수술·꽃잎·꽃받침) 있음
겉씨식물(나자식물): 씨방 없이 씨가 밖으로 노출
은행나무 → 겉씨식물 → "열매"가 아닌 "종자"라고 해야 정확
소나무 → 겉씨식물 → 암구화·수구화(배우체)로 불러야 정확

은행나무 분류 논란

  • 침엽수? 활엽수? — 잎의 형태는 활엽수처럼 넓지만 나자식물(겉씨식물)
  • 수배우체, 암배우체로 구분
  • 성 감별: DNA 분석기술로 가능

외떡잎식물과 쌍떡잎식물

구분외떡잎식물쌍떡잎식물
떡잎1장2장
뿌리수염뿌리곧은뿌리+곁뿌리
잎맥나란히맥그물맥
대표벼·대나무·야자수대부분의 나무·초화
  • 포자식물 → 종자식물: 겉씨식물(침엽수류, 은행나무) + 속씨식물(활엽수류)
  • 오떡은 쌍력임 (외떡잎식물 ≠ 쌍떡잎식물)
  • 씨방이 과육이 된다 (속씨식물)

수종잎겨울눈특징

소나무 품종·지역변이

  • 금강소나무(강송·춘양목): 경북 북부·강원 남부 분포. 줄기 곧고 재질 우수. 일제시대 춘양역을 통해 반출 → 이름에 잔재
  • 이봉용 소나무: 전북 완주 특산. 전나무 수형으로 곧게 자람
  • 반송(盤松): 줄기가 밑에서 여러 갈래로 갈라짐 (쟁반처럼 퍼짐)
  • 소나무 꽃가루(송화가루): 풍매화
  • 천연기념물 → 후계목 보전 사업

전나무·구상나무·가문비나무

전나무: 잎 끝 뾰족
구상나무: 잎 끝 오목하게 들어감. 우리나라 특산식물, 해발 1000m 이상 자생. 기후변화로 고사 급증. 솔방울: 포린(돌기)이 있어 전나무보다 예쁨

메타세쿼이아 vs 낙우송

메타세쿼이아: 잎 마주나기. 중국 원산
낙우송: 잎 어긋나기. 미국 원산. 가로수 안 되는 이유: 기근(공기뿌리)이 올라와 보도블록 뚫음

측백나무류 구별 — 기공선

측백나무: 앞뒤 모두 녹색
화백: 뒷면 흰색 X자 기공선
편백(히노키): 뒷면 흰색 Y자 기공선. 피톤치드 함량 최다

솔방울(구과) 종류

  • 구과: 솔방울 (소나무류)
  • 시과: 날개 달린 열매 — 단풍나무, 고로쇠, 느릅나무 씨앗에 날개 있음
  • 협과: 콩깍지 — 콩, 아까시나무, 회화, 등나무, 박태기, 히어리
  • 물푸레나무: 물을 푸르게 하는 나무

8. 참나무 6형제

참나무류 6종: 상수리·굴참·신갈·떡갈·갈참·졸참나무
모두 도토리 생산 → 통칭 "도토리나무". "참(眞)": 버릴 데 없는 나무
참나무 죽령개 이야기 — 상수리: 수라상, 굴참나무는 코르크가 특징
수종잎 형태잎자루잎 뒷면이름 유래서식지
상수리 긴 타원형 (피침형) ○ (15mm) 앞·뒤 동일 녹색 도토리 중 가장 맛있어 임금 수라상에 올림 → 수라→수리→상수리. 국고자목 NO! 민가 주변
굴참 긴 타원형 (피침형) ○ 흰털 밀생 (흰색) 코르크 골이 깊음. 너와집 지붕 재료 사면, 절벽
갈참 도타원형 (거꾸로) ○ — 단풍이 늦게까지 달려있음 (잎을 갈음) 산 중턱, 평탄지역
졸참 도타원형 (거꾸로) ○ — 졸병처럼 작다. 가장 맛있음 계류(산 골짜기)
떡갈 도타원형 (거꾸로) × (거의 없음) 갈색, 털있음 떡 찔 때 깔개. 두렵고 큰 잎 산불 후 개척종, 산정상, 해안
신갈 도타원형 (거꾸로) × (거의 없음) 매끈 짚신 밑창에 깔았음 산정상

이름 유래 암기

  • 상수리: 수라상 → 수리 → 상수리
  • 굴참: 코르크 골이 깊음 → 굴참
  • 신갈: 짚신 밑창에 깔았음
  • 갈참: 단풍이 늦게 달려있음 (잎을 갈음)
  • 떡갈: 떡 찔 때 깔개
  • 졸참: 졸병처럼 작다

표고버섯 원목 적합성

  • 가장 적합: 상수리나무 (균 잘 퍼지고 5~6년 사용 가능)
  • 굴참: 오래가지만 균이 잘 안 퍼짐

가시나무 (상록 참나무류)

낙엽성 참나무 6형제 ≠ 가시나무
가시나무: 남도 지방 자생 상록성 참나무류. 잎이 겨울에도 안 떨어짐. 가시없음 (가시: 일본어)

12. 기출·예상 시험문제 (12문항)

출처: 김창섭 기출문제 HWP + 음성정리 DOCX

Q1. 통용될 수 있는 하나의 과학적인 이름을 무엇이라 하는가?
A: 학명
국제어(라틴어) 표기. 생물 분류: 계·문·강·목·과·속·종 7단계
Q2. 다음 중 잣나무류에 속하는 수종은?
① 소나무 ② 백송 ③ 해송 ④ 방크스소나무
A: ② 백송
잣나무류(5엽속): 잎이 5개씩 묶음. 백송은 3엽속이지만 잣나무류에 속함. 소나무·해송(곰솔)·방크스소나무는 모두 2엽속.
Q3. 다음 중 낙엽성 참나무류가 아닌 것은?
① 신갈 ② 떡갈 ③ 굴참 ④ 가시나무
A: ④ 가시나무
참나무 6형제(상수리·굴참·신갈·떡갈·갈참·졸참)는 모두 낙엽성. 가시나무는 남도 지방 자생 상록성 참나무류.
Q4. 잎은 단엽으로 마주나고 열매는 시과(翅果: 날개 달린 형태)인 것은?
① 층층나무과 ② 진달래과 ③ 단풍나무과 ④ 버드나무과
A: ③ 단풍나무과
단풍나무류: 잎이 마주나기. 열매는 날개 달린 시과(翅果). 프로펠러처럼 생김.
Q5. 소나무에 대한 설명이 올바르지 않은 것은?
① 침엽수에 속한다. ② 두 개씩 뭉쳐난다. ③ 양성화이다. ④ 암수한그루이다.
A: ③ 양성화이다 (→ 틀린 설명)
소나무는 자웅동주(암수한그루)이지만 양성화가 아닌 단성화! 암꽃(위쪽, 빨간색)과 수꽃(아래쪽, 송화가루)이 같은 나무의 서로 다른 가지에 핌.
Q6. (주관식) 낙엽성 참나무류 중 잎은 긴 타원형이고 뒷면에 흰털이 밀생하고 줄기에 두꺼운 코르크층이 발달한 수종은?
A: 굴참나무
상수리나무와 잎 형태는 비슷하나, 뒷면이 흰색(흰털 밀생)이면 굴참나무. 코르크층 두껍게 발달 — 너와집 지붕 재료로 사용.
Q7. 다음 중 콩과에 속하지 않는 수종은?
① 아까시나무 ② 모과나무 ③ 등나무 ④ 자귀나무
A: ② 모과나무 (장미과 명자나무속)
아까시나무·등나무·자귀나무는 모두 콩과. 열매가 콩깍지처럼 생김. 모과나무는 장미과.
※ 음성정리에는 모감주나무(무환자나무과)가 정답으로 나오지만, 기출HWP에는 모과나무가 ②번 선택지로 정답 처리되어 있음. 두 버전 모두 숙지 필요.
Q8. 단풍나무 중에서 외국에서 도입된 수종은?
① 고로쇠 ② 신나무 ③ 산겨릅나무 ④ 중국단풍
A: ④ 중국단풍
신나무·한계름나무(산겨릅나무)는 우리나라 자생 단풍나무류. 중국단풍은 중국에서 도입.
Q9. (주관식) 수목 기관 중에서 광합성·호흡·증산작용을 담당하는 곳은?
A: 잎 (기공, 잎 뒷면)
기공은 광합성 시 CO₂ 흡수·O₂ 방출, 증산작용 시 수분(수증기) 배출. 잎 뒷면에 위치.
Q10. (주관식) 암꽃과 수꽃이 각각 다른 그루에 달리는 나무를 무엇이라 하는가?
A: 자웅이주(雌雄異株) / 암수딴그루
같은 나무에 암·수꽃이 달리면 자웅동주(암수한그루). 대표 자웅이주 수종: 은행나무, 버드나무류.
Q11. 측백나무과에 속하고 잎 뒷면이 Y자 형태를 나타내는 수종은?
A: 편백(히노키)
화백 = X자형 기공선 / 편백 = Y자형 기공선. 두 수종 모두 피톤치드 함량이 높아 삼림욕에 활용.
Q12. 소나무·리기다소나무·스트로브잣나무의 잎 수를 모두 더하면 몇 개인가?
A: 10개
소나무 2엽 + 리기다소나무 3엽 + 스트로브잣나무 5엽 = 10개.
